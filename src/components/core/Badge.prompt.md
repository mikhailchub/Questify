Tiny status or count badge. `soft` (tinted) by default, `solid` for high emphasis; `dot` adds a status dot.

```jsx
import { Badge } from "../core/Badge";

<Badge tone="danger" dot>Overdue</Badge>
<Badge tone="gold" variant="solid">Lv 14</Badge>
<Badge tone="success">Done</Badge>
```

Tones: neutral, gold, success, danger, info.
